# Bik Lok 

**成员：** ***徐泽斌、李松洋***

## 完成功能

- 注册、登录与访客模式
- 双击 收藏/取消收藏 并记录
- 类似抖音全屏item播放，滑动切换
- 单击视频 暂停/继续，双击蹦出桃心
